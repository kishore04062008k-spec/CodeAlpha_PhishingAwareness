# Phishing Awareness Training

## Overview

The objective of this project is to create an awareness 
training module that educates users on how phishing attacks 
work, how to recognize them, and how to stay protected 
against social engineering threats.

## Technologies & Libraries

- Node.js
- PptxGenJS
- React Icons
- Sharp

## Capabilities

- Covers all major phishing types (Email, Spear, Smishing, 
  Vishing, Whaling)
- Explains psychological tricks attackers use (urgency, 
  fear, authority, curiosity)
- Breaks down anatomy of a real phishing email
- Compares legitimate vs fake websites
- Covers real-world scam patterns people encounter daily
- Includes an interactive quiz with answers
